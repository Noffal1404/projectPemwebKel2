## Deskripsi Website
Website ini merupakan project Pemrograman Website 1 dengan tema Office Supplies Store. 
Website ini dibuat sebagai media informasi dan katalog produk alat tulis serta perlengkapan kantor yang menampilkan halaman beranda, daftar produk, detail produk, tentang kami, dan kontak.
Website dirancang dengan tampilan yang sederhana, modern, dan responsif, serta siap di-deploy secara online menggunakan platform hosting seperti Vercel.
